# 📋 Project Overview

> **Comprehensive overview of the Healthcare Lab Test Booking System.**

---

## 🎯 Project Vision

Healthcare Lab is a modern, full-stack healthcare platform designed to streamline lab test bookings, manage appointments, and deliver diagnostic reports with ease and efficiency.

---

## 🌟 Key Features

- 🔐 Secure JWT-based authentication
- 📅 Smart booking system with slot management
- 📋 Comprehensive test catalog (100+ tests)
- 📊 Real-time booking tracking
- 💳 Integrated payment gateway
- 📱 Responsive mobile-first design
- 🧪 PDF report generation
- 📈 Admin analytics dashboard

---

## 🛠️ Technology Stack

### Backend
- Java 21 LTS
- Spring Boot 3.2.2
- MySQL 8.0
- Redis 7.0
- Flyway for migrations

### Frontend
- React 19
- Vite 5.x
- TypeScript 5.x
- Tailwind CSS v4
- Framer Motion

---

## 👥 User Roles

| Role | Description |
| :--- | :--- |
| **PATIENT** | Book tests, view reports, manage appointments |
| **TECHNICIAN** | Collect samples, submit results, update status |
| **MEDICAL_OFFICER** | Verify reports, add clinical observations |
| **ADMIN** | Manage users, tests, view analytics |
