## ⚡ Quick Start

### Automated Setup (Windows)

```bash
# From the root directory
startup.bat
```

This will concurrently launch:
1. **Spring Boot Backend** (`http://localhost:8080`)
2. **React 19 Frontend** (`http://localhost:5173`)
